About The Genre Browser

The Echo Nest knows about hundreds and hundreds of genres, from A cappela to Zydeco. With The Genre Broweser you can learn about a new genres You can read about the background and history of the genre, listen to its music, learn about the genre's key artists and more.

The Genre Browser was built using the Echo Nest's Genre APIs and the Spotify Web API. You can learn more about The Echo Nest's extensive music APIs at developer.echonest.com
